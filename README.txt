README

This GitHub repository contains the experiment reported in the paper "I don't know if projection is categorical. Did Mandelkern et al. 2020 discover that it is?", as well as the analysis scripts. The relevant files for that experiment are the following:
- /experiments/main/13_explicitIgnorance: This contains the experiment code. The experiment itself can be run by loading experiment.html.
- /results/main/: This folder contains the raw data and the clean data (under /data), the R scripts used to analyze the data (under /rscripts), the generated graphs (under /graphs), and the models (under /models).
- /output/naturalness-paper: This is where the TEX code of the paper is located (as well as files to compile the paper)
